# Moodboard Personal con CSS Grid

## Autor
Paula Andrade López

## Descripción de la práctica
Esta práctica consiste en la creación de un moodboard personal utilizando HTML5 y CSS. 
La disposición de los elementos se realiza con CSS Grid siguiendo el modelo propuesto. 
El diseño es responsive mediante media queries y presenta información personal, proyectos y hobbies.

## Tecnologías utilizadas
- HTML5
- CSS3
- CSS Grid
- Media Queries
